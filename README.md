# Diabetes Prediction Using Machine Learning

This project predicts whether a patient is diabetic or non-diabetic using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Dataset
Diabetes Prediction Dataset

## Features
- Data preprocessing
- Model training
- Diabetes prediction
- Machine learning based analysis

 # Future Improvements
